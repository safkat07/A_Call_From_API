Discuss the scope of var, let, and const!!

var:
var is function-scoped, it means it can be accessed within the function where it is declared, regardless of block scope.It also has a global scope.
let:
Let is a block-scoped, it can be only accessable within the block where it is declared, such as a loop or an if statement. It allows for better control over variable scope and reduces the risk of variable pollution.
const:
const is a block-scoped, but it has it's own power.When we declare a variable as const , its value is fixed , we can not change it or reassign the value.It's good to use for values that should not change.
